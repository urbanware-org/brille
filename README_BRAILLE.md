## ⠙⠑⠞⠁⠊⠇⠎

⠞⠓⠑ ⠙⠑⠋⠊⠝⠊⠞⠊⠕⠝ ⠁⠃⠕⠧⠑ ⠁⠉⠞⠥⠁⠇⠇⠽ ⠑⠭⠏⠇⠁⠊⠝⠎ ⠊⠞ ⠁⠇⠇. ⠎⠊⠍⠏⠇⠑ ⠁⠝⠙ ⠑⠁⠎⠽ ⠞⠕ ⠥⠎⠑.

⠓⠕⠺⠑⠧⠑⠗, ⠞⠓⠑ ⠍⠕⠙⠥⠇⠑ ⠗⠑⠟⠥⠊⠗⠑⠎ ⠁ ⠎⠓⠑⠇⠇ ⠞⠓⠁⠞ ⠥⠎⠑⠎ ⠥⠞⠋-⠼⠓ ⠑⠝⠉⠕⠙⠊⠝⠛, ⠕⠞⠓⠑⠗⠺⠊⠎⠑ ⠊⠞ ⠺⠊⠇⠇ ⠉⠗⠁⠎⠓ ⠙⠥⠑ ⠞⠕ ⠑⠝⠉⠕⠙⠊⠝⠛ ⠑⠗⠗⠕⠗⠎.

⠞⠓⠑ ⠉⠕⠙⠑ ⠺⠁⠎ ⠺⠗⠊⠞⠞⠑⠝ ⠕⠝ *⠇⠊⠝⠥⠭* ⠁⠝⠙ ⠓⠁⠎ ⠕⠝⠇⠽ ⠃⠑⠑⠝ ⠞⠑⠎⠞⠑⠙ ⠕⠝ ⠞⠓⠁⠞ ⠏⠇⠁⠞⠋⠕⠗⠍, ⠽⠑⠞.

## ⠗⠑⠟⠥⠊⠗⠑⠍⠑⠝⠞⠎

### *⠚⠥⠇⠊⠁*

⠞⠓⠊⠎ ⠍⠕⠙⠥⠇⠑ ⠺⠁⠎ ⠺⠗⠊⠞⠞⠑⠝ ⠥⠎⠊⠝⠛ *⠚⠥⠇⠊⠁* ⠼⠁.⠼⠁.⠼⠚, ⠃⠥⠞ ⠎⠓⠕⠥⠇⠙ ⠁⠇⠎⠕ ⠺⠕⠗⠅ ⠺⠊⠞⠓ ⠑⠁⠗⠇⠊⠑⠗ ⠁⠝⠙, ⠕⠋ ⠉⠕⠥⠗⠎⠑, ⠝⠑⠺⠑⠗ ⠧⠑⠗⠎⠊⠕⠝⠎.

### *⠏⠽⠞⠓⠕⠝*

⠝⠕⠞⠊⠉⠑ ⠞⠓⠁⠞ ⠞⠓⠊⠎ ⠍⠕⠙⠥⠇⠑ ⠥⠎⠑⠎ ⠞⠓⠑ *⠏⠽⠞⠓⠕⠝* ⠼⠉.⠭ ⠋⠗⠁⠍⠑⠺⠕⠗⠅. ⠞⠓⠑⠗⠑ ⠊⠎ ⠝⠕ ⠕⠋⠋⠊⠉⠊⠁⠇ ⠧⠑⠗⠎⠊⠕⠝ ⠋⠕⠗ *⠏⠽⠞⠓⠕⠝* ⠼⠃.⠭, ⠃⠥⠞ ⠊⠋ ⠽⠕⠥ ⠝⠑⠑⠙ ⠞⠓⠁⠞ ⠋⠕⠗ ⠺⠓⠁⠞⠑⠧⠑⠗ ⠗⠑⠁⠎⠕⠝, ⠽⠕⠥ ⠉⠁⠝ ⠞⠗⠽ ⠗⠑⠋⠁⠉⠞⠕⠗⠊⠝⠛ ⠞⠓⠑ ⠎⠽⠝⠞⠁⠭ ⠋⠗⠕⠍ *⠏⠽⠞⠓⠕⠝* ⠼⠉.⠭ ⠞⠕ ⠧⠑⠗⠎⠊⠕⠝ ⠼⠃.⠭ ⠥⠎⠊⠝⠛ ⠞⠓⠑ *⠼⠉⠞⠕⠼⠃* ⠞⠕⠕⠇.

⠓⠕⠺⠑⠧⠑⠗, ⠞⠓⠑⠗⠑ ⠊⠎ ⠝⠕ ⠛⠥⠁⠗⠁⠝⠞⠑⠑ ⠞⠓⠁⠞ ⠞⠓⠊⠎ ⠺⠕⠗⠅⠎ ⠏⠗⠕⠏⠑⠗⠇⠽ ⠕⠗ ⠁⠞ ⠁⠇⠇.

## ⠥⠎⠁⠛⠑

### ⠑⠝⠉⠕⠙⠑ ⠁ ⠎⠞⠗⠊⠝⠛ ⠊⠝⠞⠕ ⠃⠗⠁⠊⠇⠇⠑ ⠁⠇⠏⠓⠁⠃⠑⠞

⠎⠥⠏⠏⠕⠗⠞⠑⠙ ⠉⠓⠁⠗⠁⠉⠞⠑⠗⠎ ⠁⠗⠑ ⠇⠑⠞⠞⠑⠗⠎, ⠝⠥⠍⠃⠑⠗⠎ ⠁⠝⠙ ⠎⠏⠁⠉⠑⠎. ⠁⠇⠇ ⠕⠞⠓⠑⠗⠎ ⠺⠊⠇⠇ ⠃⠑ ⠊⠛⠝⠕⠗⠑⠙. ⠞⠓⠑ ⠃⠗⠁⠊⠇⠇⠑ ⠇⠑⠞⠞⠑⠗⠎ ⠺⠊⠇⠇ ⠃⠑ ⠗⠑⠞⠥⠗⠝⠑⠙ ⠥⠎⠊⠝⠛ ⠥⠞⠋-⠼⠓ ⠉⠓⠁⠗⠁⠉⠞⠑⠗⠎.

#### *⠚⠥⠇⠊⠁*

```⠚⠥⠇⠊⠁
⠃⠗⠁⠊⠇⠇⠑ = ⠎⠞⠗⠊⠝⠛⠼⠃⠃⠗⠁⠊⠇⠇⠑("⠞⠓⠊⠎ ⠊⠎ ⠁⠝ ⠑⠭⠁⠍⠏⠇⠑")
```

#### *⠏⠽⠞⠓⠕⠝*

```⠏⠽⠞⠓⠕⠝
⠃⠗⠁⠊⠇⠇⠑ = ⠃⠗⠊⠇⠇⠑.⠎⠞⠗⠊⠝⠛⠼⠃⠃⠗⠁⠊⠇⠇⠑("⠞⠓⠊⠎ ⠊⠎ ⠁⠝ ⠑⠭⠁⠍⠏⠇⠑")
```

### ⠙⠑⠉⠕⠙⠑ ⠃⠗⠁⠊⠇⠇⠑ ⠁⠇⠏⠓⠁⠃⠑⠞ ⠊⠝⠞⠕ ⠁ ⠎⠞⠗⠊⠝⠛

⠝⠕⠞⠊⠉⠑ ⠞⠓⠁⠞ ⠽⠕⠥ ⠉⠁⠝ ⠕⠝⠇⠽ ⠥⠎⠑ ⠞⠓⠑ ⠥⠞⠋-⠼⠓ ⠉⠓⠁⠗⠁⠉⠞⠑⠗⠎ ⠞⠕ ⠑⠝⠞⠑⠗ ⠞⠓⠑ ⠃⠗⠁⠊⠇⠇⠑ ⠇⠑⠞⠞⠑⠗⠎. ⠁⠇⠇ ⠕⠞⠓⠑⠗ ⠉⠓⠁⠗⠁⠉⠞⠑⠗⠎ ⠺⠊⠇⠇ ⠃⠑ ⠊⠛⠝⠕⠗⠑⠙.

#### *⠚⠥⠇⠊⠁*

```⠚⠥⠇⠊⠁
⠎⠞⠗⠊⠝⠛ = ⠃⠗⠁⠊⠇⠇⠑⠼⠃⠎⠞⠗⠊⠝⠛("⠞⠓⠊⠎ ⠊⠎ ⠁⠝ ⠑⠭⠁⠍⠏⠇⠑"))
```

#### *⠏⠽⠞⠓⠕⠝*

```⠏⠽⠞⠓⠕⠝
⠎⠞⠗⠊⠝⠛ = ⠃⠗⠊⠇⠇⠑.⠃⠗⠁⠊⠇⠇⠑⠼⠃⠎⠞⠗⠊⠝⠛("⠞⠓⠊⠎ ⠊⠎ ⠁⠝ ⠑⠭⠁⠍⠏⠇⠑"))
```
